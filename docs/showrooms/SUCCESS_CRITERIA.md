# LaunchBase Success Criteria

## Testable Quality Gates (10 Max)

1. **Hero clarity:** Visitor understands value prop within 5 seconds
2. **CTA visibility:** "Hand It Off" button above fold on all devices
3. **Mobile responsive:** 100% functional on 320px-2560px widths
4. **Load performance:** First Contentful Paint <1.5s, Total Load <2s
5. **Accessibility:** WCAG AA compliant (color contrast, keyboard nav, screen reader)
6. **Content hierarchy:** H1 → H2 → H3 structure, no skipped levels
7. **CTA conversion path:** Maximum 2 clicks from any page to primary CTA
8. **Brand consistency:** Orange (#FF6B35) used consistently, no conflicting colors
9. **Copy clarity:** Flesch Reading Ease >60 (8th grade level)
10. **Error-free:** No console errors, no broken links, no 404s

## Automated Checks

- Lighthouse score: Performance >90, Accessibility >95, Best Practices >90, SEO >90
- HTML validation: W3C compliant
- Link checker: All internal/external links resolve
- Responsive test: Passes on Chrome, Firefox, Safari (mobile + desktop)

## Manual Review Gates

- [ ] Value prop clear without scrolling
- [ ] CTA stands out visually
- [ ] Copy flows naturally (no awkward phrasing)
- [ ] Brand voice matches brief (confident, direct, empowering)
- [ ] No jargon or technical terms without explanation

## Regression Prevention

- [ ] No changes to core value prop language
- [ ] No changes to 48-hour build promise
- [ ] No changes to orange brand color
- [ ] No reduction in accessibility score
- [ ] No increase in load time
